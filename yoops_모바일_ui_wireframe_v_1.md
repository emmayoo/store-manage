# YOOPS 모바일 UI Wireframe (Prototype v1)

> 기준: **기획안 v1**
> 목적: 디자이너·개발자가 동일한 화면을 그릴 수 있는 **로우–미드 피델리티 모바일 프로토타입**
> 디바이스 가정: iPhone 14 / Android 기본 (하단 탭 기반)

---

## 0. 글로벌 UI 구조

```
┌──────────────────────────┐
│ Header                   │
│  - 현재 매장 / 알림 🔔   │
├──────────────────────────┤
│ Content Area             │
│  (스크롤)                │
├──────────────────────────┤
│ Tab Bar                  │
│  Home | Cal | + | Store | My │
└──────────────────────────┘
```

- 알림 🔔: 우측 상단 고정 (드롭다운)
- 매장 전환: Header의 매장명 탭

---

## 1. Home (개인 통합 대시보드)

### 1-1. 와이어프레임

```
[YOOPS]            [🔔●]
──────────────────────
[ 지금 근무 중 🔴 ]
  강남점 14:00 ~ 18:00

[ 다음 근무 ]
  홍대점  내일 09:00

[ My TODO ]   [+]
  ☐ 발주 확인
  ☑ 청소

[ Today’s Notices ]
  ■ 강남점 | 발주 금지 목록
  ■ 홍대점 | 회식 공지
```

### 1-2. UI 포인트
- 카드 단위 정보
- 매장 색상 태그 항상 노출
- Home에서는 **요약만**

---

## 2. Calendar (통합 캘린더)

### 2-1. 상단

```
[일] [주] [월]
──────────────────────
[필터 ▼]   [⬇ 이미지 저장]
```

### 2-2. 필터 패널 (슬라이드)

```
기간  ───────────────
|←──────────────→|

매장
☑ 강남점  ☑ 홍대점

사람
☑ 나  ☐ 민수  ☐ 지은

[적용]
```

### 2-3. 캘린더 영역 (주간 예)

```
Mon   Tue   Wed   Thu   Fri
──────────────────────────
███        ██████
███        ██████
```

- 색 = 사람
- 이름은 시작점만 표시

## 4. Store 내부

## 6. 플로팅 액션 (+)

```

- 개인 TODO 추가
- (권한자) 공지 작성
- (권한자) 이벤트 추가
```

## 7. 디자인 가이드 (초안)

- Primary Color: 매장별 색상
- Neutral UI + Color Accent
- 카드 기반
- 아이콘 위주, 텍스트 최소화
